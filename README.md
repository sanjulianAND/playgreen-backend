## Description

PLAYGREEN Sports API is a sports betting application developed with the [Nest](https://github.com/nestjs/nest) framework. It provides a series of endpoints for managing users, bets, and transactions, following best practices of clean code and design patterns.

## Installation

```bash
$ npm install

## Development
$ npm run start

## watch mode
$ npm run start:dev

## production mode
$ npm run start:prod

## Configuration
Environment Variables
Set the following variables in the .env file:

DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=root
DB_DATABASE=playgreen_sports
JWT_SECRET=JWT_SECRET
JWT_EXPIRATION=3600

## Implemented Endpoints
User Endpoints / Permissions:
1. Place a bet on a specific event
2. Deposit money in its account (create corresponding transaction)
3. Withdraw money (create corresponding transaction)
4. Update user data
5. Request its balance (calculate balance based on transactions)
6. Get its transactions (can be filtered by type deposit, withdraw, bet, winning)
Admin Endpoints / Permissions:
1. List all of the bets (can be filtered by specific event or sport)
2. List all of the user transactions (can be also filtered by specific user and/or category)
3. Request user balance
4. Change a bet status (active / cancelled)
5. Block a specific user (user state => active / blocked) (canâ€™t block other admins)
6. Settle bets results (won / lost)
7. This settlement should trigger payments for users that have a placed bet for the winning option in case of a won
Update user data
